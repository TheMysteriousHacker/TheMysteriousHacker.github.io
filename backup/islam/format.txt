files for chapters should have names like:
chpt1, chpt2, chpt3, etc

html file for subchapter should  just have numbers:
1.html, 2.html, 3.html, etc

subchapter body in this format:
<html>
<body>
   /*text*/
</body>
</html>

<h2> element will only contain <span> if two or more lines.

<p>					normal
<p id="quote">				quote
<p id="arabic verse" verse="SNumber:VNumber">	etc .ayah3 .ayah2 .ayah1
<ol id="reference">				footnote order list
